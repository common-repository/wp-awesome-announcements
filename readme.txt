Best WordPress Announcements Plugin integrated with Custom Post Type. WP Awesome Announcements based on latest jQuery UI.

== Description ==

No need extra configurations for WP Awesome Announcements. WP Awesome Announcements allows to create unlimited Announcements Items with Title, Description. With the plugin installation's a Custom Post Type named "Announcements" will be created automatically. 

= Features Included =
* WP Awesome Announcements includes- Title, Description.
* No Post Description Limits.

== Installation ==

1. Upload `wp-awesome-announcements.zip` to the `/wp-content/plugins/` directory and extract it.
2.  From WordPress Dashboard go to "Plugins>> Add New>> Uploads", select 'wp-awesome-announcements.zip' file and upload it. Or,
3. Activate the plugin through the 'Plugins' menu in WordPress
